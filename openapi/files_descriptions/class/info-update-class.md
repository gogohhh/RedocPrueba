PUT [https://api.dynamicore.io/private/diagrams/class]

| Parameter | Required | Type | Description | Example |
| --------- | --------- | --------- | --------- |--------- |
| id | Y | Number | Id of flow | 1 |

| output | Y | Number | Id of service | 1 |
| description | Y | String | Diagram | CC077 |

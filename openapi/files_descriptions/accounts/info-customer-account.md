This endpoint is used to customer accounts.
### HTTP Request

GET [https://api.dynamicore.io/v1/accounts]

### Query Parameters

| Parameter | Required | Type | Description | Example |
| --------- | --------- | --------- | --------- |--------- |
| id | Y | String | Account Identifier | 1 |
| client_id | Y | String | Client Identifier | 1 |
